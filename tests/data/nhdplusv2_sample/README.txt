NHDPlusV2_sample
Matthew P. Geheran
10 September 2023

This directory contains test input based on the NHDPlusV2 dataset
(Horizon Systems, https://nhdplus.com/NHDPlus/index.php, accessed 10 September
2023). The shapefile, nhdplus_v2_sample.shp, contains geometry for a stream
network consisting of five reaches and the following attributes:
  - COMID
  - LENGTHKM
  - SLOPE
  - FromNode
  - ToNode
  - Hydroseq
  - DWNCOMID
